# 04 — UI/UX Wireframes

Text-based wireframes for every key page, mobile + desktop. Notation:

- `▢` = image / media
- `█` = primary CTA
- `▒` = secondary CTA
- `…` = scrollable / repeating
- `(RTL)` = layout mirrors for Arabic

## 19. RTL Notes

- All wireframes above mirror horizontally for `dir="rtl"`.
- Numerals stay LTR (Western Arabic numerals — Algerian convention).
- Currency symbols (`$`, `DA`) sit on the right of the number for RTL.
- Icons are not flipped (icons are universal — arrows ARE flipped: `→` becomes `←`).
- Form labels right-aligned for AR.

## 20. Empty / Error / Loading States

| State        | Pattern                                                                 |
|--------------|-------------------------------------------------------------------------|
| Empty listing| Illustration + "No vehicles match your filters" + Reset button         |
| Skeleton     | Vehicle cards show shimmer placeholders during AJAX                    |
| 404          | "We couldn't find that page" + search + popular vehicles               |
| 500          | "Something went wrong" + WhatsApp contact + reload button              |
| Form errors  | Inline under each field, red border, accessible aria-describedby       |
| Form success | Green check + "We received your request" + WhatsApp follow-up button   |
